Documentación

Esta aplicación permite gestionar la información de la Selección Colombia, incluyendo jugadores, equipos, partidos, torneos y plantillas. Se utiliza FastAPI para el backend, Jinja2 para plantillas, y pandas para manejar los datos en archivos CSV.

El proyecto cumple con los requisitos del curso de Desarrollo de Software, incluyendo:

  • Interfaz HTML con navegación coherente.
  • Operaciones CRUD para todos los modelos.
  • Estadísticas básicas usando pandas.
  • Documentación de planeación y diseño.
